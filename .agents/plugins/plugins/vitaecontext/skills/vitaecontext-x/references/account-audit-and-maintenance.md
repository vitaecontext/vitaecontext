# X account audit and maintenance

## Audit workflow

Use the full workflow only for broad account audits, posting-pattern reviews, or maintenance planning. For profile copy or pinned-post work, inspect profile fields, link path, pinned post, and up to 3-5 recent posts. For a standard audit, inspect up to 10 recent posts. Reserve 20-30 posts for deep account audits.

Review the account in this order:

1. profile identity and niche clarity
2. bio and link conversion path
3. pinned post and highlights quality
4. recent posting topics and format consistency
5. reply behavior and conversation depth
6. proof alignment with external surfaces
7. Premium-specific opportunities, if relevant

## Profile checklist

- display name is recognizable and clear
- handle aligns reasonably with cross-platform identity
- bio explains niche, value, and proof without stuffing
- profile image, header, and linked destination do not conflict with the niche
- pinned post supports the current professional goal

## Content checklist

- recent posts stay within a coherent topic range
- strong posts lead with a clear hook or insight
- threads are used only when the topic needs depth, not as a default ranking tactic
- media includes useful Alt Text when appropriate
- external links support real goals rather than replacing native value

## Engagement checklist

- the user replies to thoughtful comments on their own posts
- posts invite substantive replies when discussion is the goal
- replies on other accounts add value instead of farming attention
- burst posting is avoided when it creates repetitive or low-quality sessions

## Proof and consistency checklist

- profile claims match LinkedIn, GitHub, portfolio, and CV where relevant
- pinned assets and external links resolve cleanly
- expertise claims are supported by shipped work, writing, demos, or public proof
- off-niche experiments do not dominate the recent feed if authority in one domain is the goal

## Premium and capability checklist

- verify whether the user actually has Premium
- use long-form and reply-priority advice only when the feature exists and current limits are verified
- do not confuse paid boost tools with organic ranking behavior

## Maintenance rules

- refresh the pinned post when the main offer, flagship project, or best proof changes
- re-audit the bio when the user's niche or role shifts materially
- review the last 20-30 posts periodically for topic drift and weak patterns
- retire recurring low-signal formats instead of posting them out of habit

## Output format for audits

When reporting findings, group them by:

1. profile clarity issues
2. content-structure issues
3. engagement or conversation issues
4. consistency or proof issues
5. optional Premium opportunities

For each issue, name:

- affected profile element or posting pattern
- observed problem
- why it matters
- recommended fix

For bounded audits, include one `Depth note` naming the profile and post scope inspected, what was unavailable, and what a deeper pass would add.
